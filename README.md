# Pinuy Binuy Investment Analyzer API

Israeli urban renewal (Pinuy Binuy) project tracker and investment analyzer.

Tracks 92 projects across Israel with IAI/SSI calculations, market data, and opportunity scoring.

## Endpoints

- `GET /health` - Health check with DB status
- `GET /api/projects` - List all projects
- `GET /api/projects/:id` - Project details
- `GET /api/opportunities` - Investment opportunities
- `GET /api/dashboard` - Dashboard summary
