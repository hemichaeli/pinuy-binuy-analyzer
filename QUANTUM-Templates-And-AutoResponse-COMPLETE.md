## 🤖 מערכת המענה האוטומטי

### **זיהוי כוונות מתקדם**

המערכת מזהה 7 סוגי הודעות:

#### **1. תגובות חיוביות** 
```
מילים: מעוניין, כן, בטח, רוצה, מתאים, אשמח...
תגובה: "מעולה! 🎉 אני מכין לך את כל הפרטים..."
פעולה: העברה לסוכן
```

#### **2. שאלות ובקשות מידע**
```
מילים: כמה, איך, מתי, פרטים, מידע, מחיר...
תגובה: "שאלה מעולה! 🤔 חמי יענה על הכל בשיחה..."
פעולה: תיאום התקשרות
```

#### **3. שאלות מחיר**
```
מילים: מחיר, עולה, עלות, תשלום, כסף, דמי...
תגובה: "מצוין שאתה מתעניין במחירים! 💰 חמי יכין הצעה..."
פעולה: העברה מיידית לסוכן
```

#### **4. תגובות שליליות**
```
מילים: לא, לא מעוניין, אל תטריד, מחק...
תגובה: "מובן לגמרי! 👍 תודה שהקדשת זמן..."
פעולה: הסרה מהרשימה
```

#### **5. בקשות זמן**
```
מילים: מתי, זמן, תתקשר, מחר, בערב...
תגובה: "בטח! ⏰ מתי הכי נוח לך?"
פעולה: תיאום התקשרות
```

#### **6. תלונות ובעיות**
```
מילים: בעיה, תלונה, כעוס, לא עובד...
תגובה: "אני מצטער שיש בעיה! 😟 חמי יתקשר תוך 5 דקות..."
פעולה: העברה דחופה לסוכן
```

#### **7. Fallback**
```
כל השאר
תגובה: "תודה שכתבת! 📝 חמי יחזור אליך בהקדם..."
פעולה: העברה לסוכן
```

## 🎛️ מערכת הניהול

### **דשבורד ניהול שיחות**
📍 **URL:** `/api/conversations/`

#### **תכונות:**
- ✅ **צפייה בכל השיחות** - סטטוס, קטגוריה, זמן
- ✅ **פילטרים** - לפי סטטוס, טלפון, תאריך
- ✅ **היסטוריית הודעות** - כל הצ'אט המלא
- ✅ **שליחה ידנית** - התערבות של הסוכן
- ✅ **התראות** - דחוף, רגיל, נמוך
- ✅ **אנליטיקס** - נתונים מפורטים
- ✅ **ייצוא** - CSV של כל השיחות

### **API Endpoints חדשים**

#### **ניהול שיחות**
```
GET  /api/conversations - כל השיחות עם פילטרים
GET  /api/conversations/:id - צ'אט מלא של שיחה
PUT  /api/conversations/:id - עדכון סטטוס שיחה  
POST /api/conversations/:id/message - שליחה ידנית
```

#### **התראות סוכנים**
```
GET  /api/conversations/notifications - התראות ממתינות
PUT  /api/conversations/notifications/:id/handled - סימון כטופל
```

#### **בקרת מערכת**
```
GET  /api/conversations/auto-response/status - סטטוס מענה אוטומטי
POST /api/conversations/auto-response/test - בדיקת מענה
GET  /api/conversations/polling/status - סטטוס סריקה
POST /api/conversations/polling/restart - הפעלה מחדש
```

#### **אנליטיקס ודוחות**
```
GET  /api/conversations/analytics - נתונים מפורטים
GET  /api/conversations/export/conversations - ייצוא CSV
```

## 📊 Flow המענה המלא

### **שלב 1: הודעה נכנסת**
```
WhatsApp → INFORU API → Polling Service → Response Handler
```

### **שלב 2: ניתוח הודעה**
```
Response Handler:
├── זיהוי קטגוריה (positive, negative, question...)
├── בחירת תגובה מתאימה
├── רישום במסד נתונים
└── קביעת פעולה הבאה
```

### **שלב 3: מענה אוטומטי**
```
אם מתאים:
├── שליחת תגובה אוטומטית
├── רישום ההודעה
└── ביצוע פעולה (העברה/תיאום/הסרה)
```

### **שלב 4: ניהול על ידי סוכן**
```
אם נדרש:
├── יצירת התראה לסוכן
├── SMS דחוף (אם קריטי)
├── עדכון סטטוס שיחה
└── מעקב עד פתרון
```
