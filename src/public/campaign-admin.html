👥 נציגים במקביל
כל נציג יוצר מסלול עצמאי של slots לאותן שעות — כלומר N נציגים = N פגישות מקבילות בכל שעה.
Weight קובע כמה פגישות מוקצות לנציג ביחס לאחרים (2 = פי שניים).
✍️ הגדרות כנס חתימות
כנס חתימות הוא אירוע חד-פעמי עם עמדות מקבילות לכל בניין. הגדר תאריך, שעות, ומספר עמדות לכל בניין.
0 = ללא מרווח, חתימות רצופות
🏢 בניינים ועמדות
כל בניין מקבל מספר עמדות עצמאיות — כל עמדה = עורך דין שחותם במקביל.
דיירים מוקצים לבניין שלהם ורואים רק את slots של הבניין שלהם.
📅 ימים ושעות זמינות
בחר ימים ושעות שבהן ניתן לקבוע פגישות. ניתן להגדיר שעות שונות לכל יום.
